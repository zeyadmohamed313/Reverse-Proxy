# Reverse-Proxy
🛡️ Java Educational Reverse Proxy  A lightweight, scratch-built proxy featuring:  TLS Termination &amp; Custom HTTP Parsing.  Load Balancing: Round Robin &amp; Sticky Sessions.  Caching: ETag validation &amp; 304 handling.  Security: Token Bucket Rate Limiting.  Health Checks: Automatic failover/self-healing.  Logging: 3-tier (Access/System/Error).
